#### windows 常用命令
1. 查看当前目录下的文件 ->  dir
2. 跳转到当前目录下的某个文件 -> cd 目标文件名
3. 新建文件夹   -> md 文件夹名字
4. 返回上级目录 -> cd ..
5. 复制 -> copy 要复制的文件 要复制到的文件目录 （copy a.html file1）
6. 删除 -> del 要删除的文件名称（删除文件系统没有给任何提示，使用的时候需要小心，被删除的文件不会出现在回收站里面）
7. 修改文件名 -> rename 想修改的文件名 要修改成什么文件名(rename aaa.exe bbb.exe)
